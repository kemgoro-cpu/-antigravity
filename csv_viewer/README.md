# CSV Chart Viewer

CSVファイルのデータをインタラクティブなチャートで可視化するWebアプリケーションです。
複数ファイルの重ね合わせ表示やタイムシフト機能を備えています。

## 起動方法

ローカルのHTMLファイルをブラウザで開くだけで使用できます。
すべての依存ライブラリは `lib/` フォルダに同梱されているため、**インターネット接続は不要**です。

```
csv_viewer/
├── index.html        ← これをブラウザで開く
├── app.js
├── styles.css
└── lib/              ← 依存ライブラリ (同梱済み)
```

> **注意**: `index.html` をダブルクリックで直接開けます。ローカルサーバーは不要です。

## 基本操作

### 1. ファイルの読み込み

- サイドバーの **「Drag & Drop CSVs here」** エリアにCSVファイルをドラッグ＆ドロップ
- または **「Browse Files」** ボタンからファイルを選択
- 複数ファイルを同時に読み込み可能

### 2. CSVフォーマット

以下の形式のCSVに対応しています:

```csv
Time, RPM, Torque, Speed
s, rpm, Nm, km/h
0.000, 1500, 45.2, 0.0
0.001, 1502, 45.3, 0.1
...
```

- **1行目**: チャンネル名（`Time` 列が必須）
- **2行目**: 単位（省略可。`ms` の場合は自動的に秒に変換）
- **3行目以降**: 数値データ

> ヘッダー行の位置はサイドバーの **Settings** セクション（Name Row / Unit Row）で変更できます。
> `Time` 列が見つからない場合は最初の列が時間軸として使用されます。

### 3. チャンネルの選択

ファイル読み込み後、サイドバーの **Channels** セクションにチャンネル一覧が表示されます。

- チャンネル名をクリックして **表示 / 非表示** を切り替え
- 上部の検索ボックスでチャンネルを絞り込み可能
- 選択中のチャンネルには **Y軸範囲** (min / max) を個別設定可能

## 複数ファイルの重ね合わせ

- 最初に読み込んだファイルが **Main (M)** として設定されます
- 追加で読み込んだファイルは **Sub (S)** として破線で重ね描画されます
- ファイル一覧のバッジ (`M` / `S`) をクリックすると Main を切り替えられます
- チャンネル一覧は Main ファイルのものが表示されます（同名チャンネルが Sub にあれば自動で重ね表示）

### タイムシフト

Sub ファイルの時間軸をずらして波形を合わせることができます。

- **手動**: ファイル一覧の **Δt (s)** 入力欄にオフセット値を入力
- **自動**: **Auto** ボタンで Main との RMSE 最小化により自動アライメント（選択中のチャンネルが対象）
- **ドラッグ**: ツールバーの **Time Shift** ボタンを有効にし、チャート上をドラッグして直感的にシフト

## チャート操作

| 操作 | 方法 |
|---|---|
| **X軸ズーム** | マウスホイール / 下部スライダー |
| **X軸パン** | 下部スライダーをドラッグ |
| **Y軸ズーム** | 各チャートの右端スライダー |
| **ボックスズーム** | ツールバーの **Box Zoom** → チャート上をドラッグ |
| **ズームリセット** | ツールバーの **Reset View** |
| **値の確認** | チャート上にマウスホバー（ツールチップ表示） |

### サンプリングモード

ツールバーのドロップダウンで切り替え可能:

- **LTTB (fast)**: 大量データの描画を高速化（デフォルト）
- **None (exact)**: 全データ点をそのまま描画

## カスタム RAM（計算チャンネル）

既存チャンネルの四則演算で新しいチャンネルを追加できます。

1. サイドバーの **Custom RAM** セクションに **Name** と **式** を入力
2. **Add** ボタンをクリック

### 式の例

| 式 | 説明 |
|---|---|
| `RPM / 60` | RPM を rps に変換 |
| `Torque * RPM * 3.14159 / 30` | 出力 (W) を計算 |
| `(Speed - 50) / 10` | 正規化 |

- 使用可能な演算子: `+`, `-`, `*`, `/`, `()`
- チャンネル名はCSVのヘッダーと完全一致で指定

## その他

- **Clear All**: 全ファイルと選択状態をリセット
- ファイル一覧の **×** ボタンで個別にファイルを削除可能
